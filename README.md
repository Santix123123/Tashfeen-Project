<h1 align="center">Recipe Reader and Searcher</h1>

<h2 align="center">Replication Instructions</h2>

<h2>1. Requirements:</h2>

Make sure the following are installed:

- Python **3.9 or newer**

Required Python packages:
- numpy
- flask
- scikit-learn

Install them using:
```powershell
pip install numpy flask scikit-learn
```

<h2>2. Project Files</h2>

Place the following files in the same folder:
- content/recipes folder (containing first source of dataset)
- RecipeReader.py
- recipe_reader_webpage.html
- tashfeen.json (containint second source of dataset)

<h2>3. Running the Project</h2>

Open a terminal inside the project folder and run:
```powershell
python RecipeReader.py
```

If the program starts correctly you should see:

Running on http://127.0.0.1:5000

<h2 align="center">Click on the server address and HAVE FUN!</h2>
